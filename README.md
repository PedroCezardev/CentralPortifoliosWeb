# 🚀 Linkdperfil

> **Sua vitrine profissional em um único link.**
> O Linkdperfil é uma plataforma de "Link in Bio" personalizada, desenvolvida para profissionais que precisam centralizar portfólio, serviços e redes sociais em uma interface otimizada e de alta conversão.

---

## 📸 Sobre o Projeto

O **Linkdperfil** nasceu da necessidade de profissionais apresentarem seus trabalhos de forma mais robusta que os agregadores de links comuns. O foco aqui é **performance**, **design limpo** e **experiência do usuário (UX)**.

Este projeto foi apresentado em meus conteúdos de redes sociais (Instagram/TikTok) como um estudo de caso de:
* **Desenvolvimento Full-Stack moderno.**
* **Implementação de boas práticas de QA e Automação.**
* **Otimização de conversão de leads para freelancers.**

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no ecossistema web para garantir velocidade e escalabilidade:

* **Frontend:** React.js, Vite (com SASS).
* **Deploy:** Vercel.
* **Domínio:** Configurado via Registro.br.

---

## 🚀 Como Executar o Projeto

1. **Clone o repositório:**
   \`\`\`bash
   git clone https://github.com/seu-usuario/linkdperfil.git
   \`\`\`
2. **Instale as dependências:**
   \`\`\`bash
   npm install
   \`\`\`
3. **Inicie o servidor:**
   \`\`\`bash
   npm run dev
   \`\`\`
